Papers

- SWE-rebench V2 introduces a language-agnostic pipeline that mines real PRs into executable, containerized coding tasks (32K+ full tasks and 120K+ lightweight tasks) to scale RL training beyond Python-only setups. → [SWE-rebench V2](https://arxiv.org/abs/2602.23866)
- CUDA Agent frames CUDA kernel optimization as large-scale agentic RL and reports strong speedups against compiler baselines and competitive coding models. → [CUDA Agent: Large-Scale Agentic RL for High-Performance CUDA Kernel Generation](https://www.alphaxiv.org/abs/2602.24286)
- Agentic Code Reasoning studies whether agents can reason about repository semantics without execution and proposes semi-formal reasoning to improve structured code understanding. → [Agentic Code Reasoning](https://www.alphaxiv.org/abs/2603.01896)
- BeamPERL shows parameter-efficient RL with verifiable rewards can specialize compact LLMs for structured technical reasoning, reinforcing the code-RL pattern of objective, checkable feedback. → [BeamPERL: Parameter-Efficient RL with Verifiable Rewards](https://arxiv.org/abs/2603.04124)
- OpenEnv in Practice argues that practical tool-using agent evaluation requires real-world, executable environments rather than static offline tests, aligning benchmark design with deployment reality. → [OpenEnv in Practice: Evaluating Tool-Using Agents in Real-World Environments](https://huggingface.co/blog/openenv-turing)
